                       核心模块 (6个核心文件)
================================================================================

1. backtester_engine.py (12.5 KB)
   ├─ 高速回测引擎
   ├─ VectorBT 集成 (100倍加速)
   ├─ Polars 向量化计算
   ├─ 多进程支持
   └─ 核心类: BacktestEngine, Order, Trade
   🎯 主要用途: 极速回测，支持1000根K线<1秒

2. backtester_core.py (18.5 KB)
   ├─ 完整回测流程管理
   ├─ 数据加载和缓存
   ├─ 15+ 种特征计算
   ├─ 信号生成
   ├─ 风险管理 (止损、止盈)
   └─ 核心类: Backtester, BacktestConfig, BacktestResult
   🎯 主要用途: 完整策略回测

3. backtester_optimizer.py (20.8 KB)
   ├─ 5种参数优化算法
   ├─ 网格搜索 (GridSearch)
   ├─ 随机搜索 (RandomSearch)
   ├─ 遗传算法 (GeneticOptimizer)
   ├─ 粒子群算法 (PSOOptimizer)
   ├─ 模拟退火 (SimulatedAnnealingOptimizer)
   └─ 核心类: ParameterOptimizer, ParameterSpace
   🎯 主要用途: 参数优化，找最优超参

4. backtester_walkforward.py (15.1 KB)
   ├─ 走测验证系统
   ├─ 4种数据分割方法
   │  ├─ Rolling (滚动窗口)
   │  ├─ Anchored (锚定)
   │  ├─ Purged (净化)
   │  └─ Embargo (禁入)
   ├─ 稳定性检查
   ├─ IS/OOS 性能对比
   └─ 核心类: WalkForwardValidator, WalkForwardResult
   🎯 主要用途: 验证策略鲁棒性，避免过拟合

5. backtester_analytics.py (11.7 KB)
   ├─ 性能分析系统
   ├─ 20+ 财务指标计算
   │  ├─ Sharpe/Sortino/Calmar 比率
   │  ├─ 最大回撤
   │  ├─ 胜率和收益因子
   │  ├─ VaR/CVaR
   │  └─ 更多...
   ├─ 报告生成
   └─ 核心类: PerformanceAnalytics, PerformanceMetrics
   🎯 主要用途: 完整性能评估

6. backtester_visualizer.py (12.4 KB)
   ├─ 专业可视化
   ├─ 权益曲线图
   ├─ K线图 + 交易信号
   ├─ 回撤图 (水下图)
   ├─ 分布图 (收益/P&L)
   ├─ 热力图 (相关性/月度)
   ├─ 仪表板 (多图组合)
   └─ 核心类: BacktestVisualizer
   🎯 主要用途: 结果可视化，生成报告

总计: 2,750+ 行核心代码，20+ 个类，100+ 个函数
