# 🚀 快速开始卡片

## 一分钟启动

### Windows 用户
```
1. 双击: 启动系统.bat
2. 等待 20-30 秒
3. Electron 应用自动打开
4. 开始交易！
```

### macOS / Linux 用户
```
1. 打开终端
2. cd 塞子
3. python RUN.py
4. 等待应用启动
5. 开始交易！
```

---

## 系统就绪检查

```bash
python 系统检查.py
```

所有项都应显示 ✅

## 登录 Hyperliquid

1. 打开 Electron 应用
2. 在"交易面板"中点击 "Connect"
3. 用 Hyperliquid App 扫描二维码
4. 完成授权
5. 自动登录成功

---

## 开始交易

1. 配置交易参数 (可选)
2. 在右侧"⚙️ 设置"启用功能
3. 点击绿色"🚀 开始交易"按钮
4. 系统自动分析行情并下单

## 紧急停止

- 点击红色"⏹️ 停止交易"按钮
- 或在终端按 CTRL+C

---

## 故障排查

| 问题 | 解决 |
|------|------|
| 应用未打开 | 等待 20-30 秒 |
| 后端错误 | 重启应用 |
| 无法下单 | 检查账户余额 |
| npm 错误 | 自动使用 node 启动 |

## 配置文件

**环境变量** (.env):
```
HYPERLIQUID_API_KEY=your_key
HYPERLIQUID_SECRET=your_secret
TRADING_MODE=paper  # 改为 live
```

**交易参数** (main_complete.py):
```python
MAX_LEVERAGE = 3
KELLY_COEFFICIENT = 0.55
MIN_SIGNAL_CONFIDENCE = 0.75
RISK_LIMIT_DAILY = 0.10
```
