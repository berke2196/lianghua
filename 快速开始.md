# ⚡ 5分钟快速开始指南

## 前置条件
- ✅ Python 3.8+
- ✅ Hyperliquid 账户
- ✅ API Key & Secret

---

## 第一步：获取 API Key (2分钟)

1. 访问 https://hyperliquid.xyz
2. 登录账户
3. **Settings** → **API Management**
4. **Create New Key**
5. 复制 **API Key** 和 **API Secret**

---

## 第二步：启动系统 (1分钟)

### Windows
```bash
# 双击运行
run.bat
```

### Mac/Linux
```bash
bash run.sh
```

### 或手动启动
```bash
pip install fastapi uvicorn aiohttp numpy
python main.py
```

等待看到：
```
✅ FastAPI 已启动
🌐 访问地址: http://localhost:8000
```

---

## 第三步：登录并交易 (2分钟)

### 方式 A：WalletConnect (推荐)
1. 打开 http://localhost:8000
2. 点击 **🌐 打开 Hyperliquid 登录页面**
3. 在 Hyperliquid 上登录
4. 返回页面，点击 **✅ 检测登录状态**
5. ✅ 登录完成！

### 方式 B：手动登录
1. 打开 http://localhost:8000
2. 输入 API Key 和 Secret
3. 点击 **✅ 已登录**
4. ✅ 登录完成！

---

## 第四步：启动自动交易

### 快速开始 (推荐设置)
1. 在 **🤖 自动交易引擎** 面板
2. 交易对：**BTC** (或 ETH/SOL)
3. 策略：**📈 Momentum** (最快见效)
4. 点击 **🚀 启动自动交易**

### 等待
- 系统会每 30 秒分析一次
- 自动生成买卖信号
- 实时显示盈亏

## 常见问题

### Q1: 需要一直开着浏览器吗？
**A**: 不需要，后端会自动运行。但建议保持开窗以监控。

### Q2: 可以用多少倍杠杆？
**A**: 建议 1-3 倍。系统默认 1 倍。可在下单时修改。

### Q3: 止损是多少？
**A**: 系统自动 5% 止损，10% 止盈。可在 trading_engine.py 中修改。

### Q4: 交易对支持哪些？
**A**: BTC、ETH、SOL 等所有 Hyperliquid 支持的交易对。

### Q5: 多少钱可以开始？
**A**: 建议最少 100 USDT。先用小额测试。
