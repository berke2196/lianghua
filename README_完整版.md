## 📖 使用说明

### 第一步：登录
1. 打开 http://localhost:8000
2. 点击 **🌐 打开 Hyperliquid 登录页面**
3. 在 Hyperliquid 官网完成登录/授权
4. 返回本页面
5. 点击 **✅ 检测登录状态**
6. 显示仪表板 ✅

### 第二步：手动交易

#### 下单
1. 在 **💰 下单交易** 面板
2. 选择交易对 (BTC/ETH/SOL)
3. 选择交易类型 (买入/卖出)
4. 输入数量
5. 输入价格 (留空=市价单)
6. 设置杠杆倍数
7. 点击 **🎯 执行交易**

#### 查看持仓
- 在 **📊 当前持仓** 面板查看
- 实时显示盈亏
- 点击 **平仓** 按钮关闭持仓

#### 查看交易历史
- 在 **📝 交易历史** 面板查看
- 显示最近 10 笔交易
- 包含交易时间、价格、状态

### 第三步：自动交易

#### 启动自动交易
1. 在 **🤖 自动交易引擎** 面板
2. 选择交易对
3. 选择策略：
   - **📈 Momentum** - 快速见效，波动大
   - **↔️ Mean Reversion** - 稳定高效，适合震荡
   - **🔁 Trend Following** - 稳健策略，跟踪趋势
4. 点击 **🚀 启动自动交易**

#### 实时监控
- **状态** - 运行中/已停止
- **总交易** - 执行的交易数
- **胜率** - 盈利交易比例
- **总盈亏** - 累计盈亏
- **活跃持仓** - 当前持仓数

#### 停止自动交易
- 点击 **⏹️ 停止** 按钮
- 系统会平仓所有持仓
- 停止新的交易信号

## 📊 策略详解

### 🟢 Momentum (动量策略)

**原理**：
- RSI < 30 → 超卖 → 买入信号
- RSI > 70 → 超买 → 卖出信号
- MACD 穿叉 → 趋势信号

**适合场景**：
- 强势上升/下跌的市场
- 高波动性币种
- 短期快速交易

**风险**：
- 可能被套
- 需要及时止损

---

### 🔄 Mean Reversion (均值回归)

**原理**：
- 布林带上轨突破 → 卖出信号
- 布林带下轨突破 → 买入信号
- 价格回归中线

**适合场景**：
- 震荡市场
- 横盘整理
- 稳定波动范围

**风险**：
- 单边行情失效
- 需要充足资金

---

### 🔁 Trend Following (趋势跟踪)

**原理**：
- SMA10 > SMA20 > SMA50 → 多头排列 → 买入
- SMA10 < SMA20 < SMA50 → 空头排列 → 卖出
- 跟踪明确趋势

**适合场景**：
- 单边上升/下跌
- 长期趋势交易
- 稳健保守策略

**风险**：
- 反应较慢
- 可能滞后

POST /api/auto-trading/stop - 停止自动交易

## 🚨 重要提示

### ⚠️ 风险管理
1. **先从小额开始** - 不要一开始就用大资金
2. **设置止损** - 系统会自动止损，但要人工确认
3. **监控日志** - 定期查看交易日志
4. **不要过度杠杆** - 建议1-3倍杠杆
5. **定期检查** - 自动交易不是完全无人值守

### 🔒 安全提示
1. **妥善保管 API Key** - 不要分享给他人
2. **使用子账户** - 建议用 Hyperliquid 子账户交易
3. **限制 API 权限** - 只给予必要权限
4. **定期审计** - 检查交易日志是否异常

### 💡 最佳实践
1. **回测策略** - 先在历史数据上测试
2. **小额测试** - 新策略先用最小资金测
3. **分散投资** - 同时运行多个策略
4. **定期优化** - 根据市场调整参数
5. **备份数据** - 保存交易记录

## 🐛 故障排除

### 问题：无法连接到 Hyperliquid API
**解决**：
1. 检查 API Key 和 Secret 是否正确
2. 确认账户有足够资金
3. 检查网络连接

### 问题：自动交易没有下单
**解决**：
1. 检查账户余额
2. 检查交易对名称
3. 查看日志中的错误信息

### 问题：持仓信息不更新
**解决**：
1. 刷新页面
2. 重新登录
3. 检查后端服务是否运行
