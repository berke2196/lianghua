# ⭐ AsterDex 自动交易系统 - 桌面应用

## 🚀 快速开始

### 方式1: 直接运行（推荐）
```bash
run_app.bat
```

### 方式2: 命令行启动
```bash
python app.py
```

### 方式3: 打包成.exe（可分发）
```bash
build.bat
```

---

## 💻 系统要求

- **操作系统**: Windows 7 及以上
- **Python**: 3.8+（如果直接运行）
- **内存**: 最少 512MB
- **网络**: 需要网络连接

---

## 📦 功能模块

### 1️⃣ **登录**
- 输入 AsterDex API Key 和 API Secret
- 自动验证登录信息
- 支持多个账户切换

### 2️⃣ **交易面板** 💰
- 选择交易对 (BTCUSDT, ETHUSDT, SOLUSDT)
- 选择买入/卖出
- 设置数量、价格、杠杆
- 一键下单

### 3️⃣ **持仓管理** 📊
- 实时查看所有持仓
- 显示开仓价、当前价、未实现盈亏
- 盈亏百分比用颜色标记（绿色/红色）
- 快速平仓操作

### 4️⃣ **交易历史** 📝
- 查看最近交易记录
- 交易ID、方向、价格、时间
- 已实现盈亏统计

### 5️⃣ **自动交易** 🤖
选择3种策略之一：

#### 📈 Momentum (动量策略)
- 基于 RSI 和 MACD 指标
- 适合高波动性市场
- 快速反应市场变化

#### ↔️ Mean Reversion (均值回归)
- 基于布林带指标
- 适合震荡市场
- 稳定高效

#### 🔁 Trend Following (趋势跟踪)
- 基于移动平均线
- 适合单边趋势
- 稳健保守

**控制面板:**
- 🚀 启动 - 开始自动交易
- ⏹️ 停止 - 立即停止并平仓
- 📊 统计 - 实时显示总交易、胜率、总盈亏

---

## 🔑 API Key 获取

1. 访问 https://www.asterdex.com
2. 登录账户
3. 进入 **设置** → **API**
4. 创建新的 API Key
5. 复制 **Key** 和 **Secret** 到登录窗口

---

## ⚙️ 配置参数

编辑 `trading_engine.py` 中的 `AutoTradingEngine` 类：

```python
self.max_position_size = 0.1      # 单仓最多占资金10%
self.stop_loss_percent = 0.05     # 止损5%
self.take_profit_percent = 0.10   # 止盈10%
```

---

## 🚨 风险提示

1. **小额测试** - 先用最小资金测试
2. **设置止损** - 系统自动执行止损止盈
3. **不要过度杠杆** - 建议1-3倍
4. **定期监控** - 不是完全无人值守

---

## 🐛 故障排除

### 无法连接 API
- ✅ 检查 API Key/Secret 是否正确
- ✅ 检查网络连接
- ✅ 确认账户有足够资金

### 自动交易无法启动
- ✅ 检查账户余额
- ✅ 检查交易对是否支持
- ✅ 查看日志获取详细信息

### 软件无法启动
```bash
pip install PyQt6 aiohttp numpy requests -q
python app.py
```

## 📊 技术架构

```
app.py (PyQt6 GUI)
    ↓
asterdex_api.py (API 封装)
    ↓
trading_engine.py (策略引擎)
    ↓
AsterDex 交易所 API
```
