## 📊 5大交易算法

### 1️⃣ **做市商算法 (Market Making)** - 权重 30%
**稳定高频套利**

- 📈 日收益: 0.3-0.8% (最稳定)
- 🎯 原理: 赚取买卖价差 (Bid-Ask Spread)

**工作原理:**
```
1. 实时获取市场买卖价 (Best Bid/Ask)
2. 计算中价 (Mid Price) = (Bid + Ask) / 2
3. 计算最优点差:
   • 深度因子: 委托簿深度
   • 波动率因子: 价格波动程度
   • 频率因子: 成交频率
4. 生成买卖挂单:
   • 买价 = 中价 - 点差/2
   • 卖价 = 中价 + 点差/2
5. 头寸管理:
   • 时间止损: >30秒自动平仓
   • 利润锁定: 赚0.1%快速收益立即平仓
   • 风险止损: -0.05%立即平仓
```

**Kelly资金管理:**
- 胜率: 60-70%
- 盈亏比: 1:1
- Kelly准则: f* = 15% (保守系数÷2)

---

### 2️⃣ **统计套利算法 (Statistical Arbitrage)** - 权重 25%
**低风险配对交易**

- 📈 日收益: 0.1-0.5% (低风险)
- 🎯 原理: 发现币种间的价格偏离

**工作原理:**
```
1. 监测两个相关币种 (如 BTC/ETH)
2. 计算历史价格比率 (最近1000根K线)
3. 计算统计指标:
   • 平均比率
   • 标准差
   • Z-Score = (当前比 - 平均比) / 标准差
4. 信号生成:
   • Z-Score > 2.5: 配对A相对贵, 卖A买B
   • Z-Score < -2.5: 配对A相对便宜, 买A卖B
5. 平仓:
   • 当Z-Score回到 ±0.5时平仓 (利润/止损)
```

**优势:**
- 相关币种，风险相对对冲
- 胜率 > 55%

---

### 3️⃣ **趋势跟踪算法 (Trend Following)** - 权重 25%
**捕捉大趋势**

- 📈 日收益: 0.5-2% (当有大趋势时)
- 🎯 原理: 识别并跟踪强势趋势方向

**工作原理:**
```
1. 多时间框趋势检测 (3个时间框):
   • 短期: 当前价 vs SMA(10)
   • 中期: SMA(10) vs SMA(30)
   • 长期: 当前价 vs (最高+最低)/2
   
2. 只有三个趋势都一致才确认趋势:
   ✅ 全部UP → 做多
   ✅ 全部DOWN → 做空
   ❌ 不一致 → 观望

3. 趋势强度计算 (MACD/ATR):
   • MACD: 快速移动平均差
   • ATR: 平均真实范围
   • 强度 = |MACD/ATR| * 50
   
4. 信号生成:
   • 趋势UP + 强度>30 → 做多
   • 趋势DOWN + 强度>30 → 做空
   • 强度 < 30 → 观望
```

**特点:**
- 减少假信号 (三框一致)
- 强势趋势才交易 (强度>30)

---

### 4️⃣ **资金费率套利 (Funding Rate Arb)** - 权重 10%
**无风险收益**

- 📈 年收益: 10-50% (无风险对冲)
- 🎯 胜率: 99% (完全对冲)
- 🎯 原理: 收取资金费率 (期货特有)

**工作原理:**
```
1. 获取资金费率 (每8小时结算):
   • Hyperliquid 会定期支付或收取费率
   
2. 年化收益计算:
   • 年化收益 = 资金费率 × 3 × 365
   • (8小时一个周期，所以×3)
   
3. 判断是否值得:
   • 年化收益 >= 5% 时执行
   
4. 执行方案:
   • 现货: 长期持有
   • 期货: 反向做空 (对冲)
   • 利润: 收取资金费率
   
5. 持仓时间:
   • 一个周期 = 8小时
   • 然后重新评估
```

**优势:**
- 完全对冲风险
- 无需判断方向
- 稳定收入

---

### 5️⃣ **技术指标策略 (Technical Indicators)** - 权重 10%
**传统指标信号**

- 📈 日收益: 0.2-0.6%
- 🎯 原理: 组合使用多个技术指标

**使用的指标:**

**RSI (相对强弱指数)**
- RSI < 30: 超卖 → 做多
- RSI > 70: 超买 → 做空

**MACD (指数平滑移动平均)**
- 黄金交叉: EMA12 > EMA26 → 做多
- 死亡交叉: EMA12 < EMA26 → 做空

**布林带 (Bollinger Bands)**
- 价格 > 上轨: 强阻力 → 做空
- 价格 < 下轨: 强支撑 → 做多

**成交量 (Volume)**
- 成交量 > 平均2倍: 趋势确认

## 🔄 信号融合引擎 (Signal Fusion)

### 工作流程

```
算法1 → 信号1
算法2 → 信号2  ──→ 融合引擎 ──→ 最终决策
算法3 → 信号3
...

融合规则:
1. 提取有效信号 (非None)
2. 计算加权平均强度
3. 判断方向一致性
4. 应用风控检查
5. 生成最终交易指令
```

### 加权规则

| 算法 | 权重 | 理由 |
|------|------|------|
| 做市商 | 30% | 最稳定高频 |
| 统计套利 | 25% | 低风险平衡 |
| 趋势跟踪 | 25% | 抓大趋势 |
| 资金费率 | 10% | 无风险补充 |
| 技术指标 | 10% | 辅助确认 |

## 🛡️ 风险管理 (3层防线)

### 第1层: 订单级别
- 最大订单大小: 账户余额的10%
- 最小订单价值: $100

### 第2层: 头寸级别
- 最大单币种持仓: 账户的20%
- 最多同时持仓币种: 5个

### 第3层: 账户级别
- 每日最大亏损: -5%
- 每月最大亏损: -15%

## 🔐 止损/止盈机制

**做市商:**
- ⏱️ 时间止损: 30秒无利润平仓
- 💰 利润锁定: 赚0.1%立即平仓
- 🛑 风险止损: -0.05%立即平仓

**趋势跟踪:**
- 🛑 ATR止损: 当前价 - ATR(14)
- 🎯 移动止损: 跟踪最高价下方2%

**统计套利:**
- 📊 Z-Score止盈: 回到±0.5
- 🛑 最大亏损: -1%退出
