## ⚙️ 配置说明

### 算法参数

| 参数 | 范围 | 说明 |
|------|------|------|
| 策略模式 | - | 选择交易策略 |
| 杠杆倍数 | 1x-10x | 提高收益同时增加风险 |
| 每笔风险 | 0.1%-5% | 单笔交易最大亏损比例 |
| 最大持仓 | 1-10 | 同时持有币种数量 |

### 风险等级

- **保守** (低风险): 杠杆2x, 每笔风险0.5%, 做市策略
- **稳健** (中风险): 杠杆3x, 每笔风险1%, 混合策略
- **激进** (高风险): 杠杆5-10x, 每笔风险2-5%, 高频策略

### Hyperliquid API

系统会自动通过 **二维码登录** 连接到您的 Hyperliquid 账户：
1. 启动应用后，会弹出二维码
2. 用 Hyperliquid App 扫描二维码
3. 授权后自动登录，开始交易

## 🔌 API 集成

### 后端 REST API

```
POST   /api/trading/start        - 启动交易
POST   /api/trading/stop         - 停止交易
GET    /api/trading/status       - 获取交易状态
GET    /api/market/{symbol}      - 获取行情数据
GET    /api/account/balance      - 获取账户余额
GET    /api/positions            - 获取持仓列表
POST   /api/order/place          - 下单
POST   /api/order/cancel         - 撤单
```

### WebSocket 数据推送

```javascript
// 行情订阅
ws://localhost:8000/ws/market/{symbol}

// 推送内容
{
  "price": 45000,
  "24h_change": 2.5,
  "volume": 1500000,
  "bid": 44999,
  "ask": 45001,
  "timestamp": 1234567890
}
```

### 实盘交易安全

1. **小额测试**: 先用 $100-$500 测试系统
2. **风控检查**: 确保风控参数正确设置

### 常见问题

**Q: 应用启动很慢?**
- 第一次启动会安装依赖,需要 1-3 分钟
- 后续启动会很快

**Q: 无法连接到后端?**
- 检查 Python 环境是否正常
- 查看后台是否有错误日志

**Q: 交易没有执行?**
- 检查 Hyperliquid API 认证是否成功
- 查看账户余额是否足够
- 检查风控是否触发

**Q: 如何查看详细日志?**
- 打开 `logs/` 目录查看日志文件
- 使用 Windows 事件查看器查看系统日志
