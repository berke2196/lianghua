## ⚠️ 重要提示

### 首次启动
- 需要 20-30 秒才能完全加载
- 等到 Electron 窗口打开再操作
- 如果 React 未启动，再等待 10-15 秒

### 登录 Hyperliquid
1. 在交易面板中点击 "Connect" 按钮
2. 使用 Hyperliquid 应用扫描二维码
3. 完成授权

### 启用自动交易
1. 在右侧"设置"中启用高频交易
2. 点击绿色"🚀 开始交易"按钮
3. 系统会自动分析行情并下单

### 防火墙
- 如果应用无法启动，检查以下端口是否被防火墙阻止:
  - 3000 (React 开发服务器)
  - 8000 (FastAPI 后端)

## 🐛 故障排查

### 问题 1: Electron 窗口不显示
**解决:**
1. 等待 15-20 秒
2. 检查 React 服务器是否已启动 (http://localhost:3000)
3. 重启应用: `Ctrl+C` 然后重新运行 `python RUN.py`

### 问题 2: 后端 API 连接失败
**解决:**
1. 检查 FastAPI 是否启动 (http://localhost:8000 应返回 API 文档)
2. 确保没有其他进程占用 8000 端口
3. 重启系统: `Ctrl+C` 然后重新运行

### 问题 3: 交易无反应
**解决:**
1. 确认后端 API 已连接 (UI 右上角显示 "✅ 后端已连接")
2. 确认已登录 Hyperliquid 账户
3. 检查账户余额是否足够
4. 查看浏览器控制台是否有错误 (F12 打开)

### 问题 4: npm 未安装
**解决:**
1. 系统会自动使用 node 启动前端
2. 如果仍失败，需要安装 Node.js: https://nodejs.org/
3. 然后运行 `npm install` 安装依赖
