【快速开始 - 只需3步】

1️⃣ 双击启动
   └─ run_app.bat

2️⃣ 输入登录信息
   └─ API Key 和 API Secret
   └─ 从 https://www.asterdex.com 获取

3️⃣ 开始交易
   └─ 手动下单
   └─ 或启动自动交易


【文件说明】

📍 run_app.bat
   └─ 一键启动应用（推荐）

📍 build.bat
   └─ 打包成 .exe 文件（可分发给他人）

📍 app.py
   └─ 应用主程序（Python源代码）

📍 使用说明_桌面应用.md
   └─ 详细功能说明和配置指南


【功能特性】

✅ 实时交易面板
✅ 持仓管理
✅ 交易历史查看
✅ 3种自动交易策略
   - 动量策略 (Momentum)
   - 均值回归 (Mean Reversion)
   - 趋势跟踪 (Trend Following)
✅ 风险管理（止损/止盈）
✅ 支持做空做多
✅ 最高10倍杠杆


【系统要求】

- Windows 7 或更高版本
- Python 3.8+ （已自动检测）
- 网络连接

第一次运行时会自动安装依赖库。


【问题解决】

❌ 点击 run_app.bat 没反应
   → 右键 → 编辑，检查Python路径是否正确
   → 或打开cmd，输入: python app.py

❌ 应用启动报错
   → 手动安装依赖:
     pip install PyQt6 aiohttp numpy requests -q
   → 再次运行 run_app.bat

❌ 无法连接API
   → 检查API Key是否正确
   → 检查网络连接
   → 确认账户有资金


【下一步】

1. 阅读 使用说明_桌面应用.md （详细功能说明）
2. 从 https://www.asterdex.com 获取 API Key
3. 双击 run_app.bat 启动应用
4. 输入登录信息
5. 开始交易！
