🔧 配置步骤:

1. 创建 .env 文件 (复制 .env.example)

2. 配置 Hyperliquid API 密钥:
   HYPERLIQUID_API_KEY=your_api_key
   HYPERLIQUID_SECRET=your_secret_key

3. 设置交易模式:
   TRADING_MODE=paper  # 测试模式
   # 改为: TRADING_MODE=live  # 实盘模式

4. 调整交易参数 (main_complete.py):
   MAX_LEVERAGE = 3
   KELLY_COEFFICIENT = 0.55
   MIN_SIGNAL_CONFIDENCE = 0.75
   RISK_LIMIT_DAILY = 0.10
