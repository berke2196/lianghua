## 5大算法简要

### 1️⃣ 做市商 | 稳定高频 | 日赚 0.3-0.8%
**赚取买卖点差**
```
买单: $44999.5  ←→  卖单: $45000.5
  ↓                      ↓
客户买        获利$1      卖给客户
```
- 高频交易，稳定收益
- 持仓时间: 30秒以内
- Kelly资金管理: 15%

---

### 2️⃣ 统计套利 | 低风险对冲 | 日赚 0.1-0.5%
**配对币种价格偏离交易**
```
监测 BTC/ETH 历史比率
当前比率偏离2.5倍标准差时:
  卖贵的 + 买便宜的
  等待回到均值时平仓
```
- 风险相对对冲
- 胜率: 70%+
- 持仓时间: 1-24小时

---

### 3️⃣ 趋势跟踪 | 捕捉大波段 | 日赚 0.5-2%
**多框一致，跟踪强势趋势**
```
短期 UP + 中期 UP + 长期 UP
     ↓           ↓           ↓
  (价>MA10)  (MA10>MA30)  (价>中点)
     
      → 全部一致才下单 → 强度>30才交易
```
- 减少假信号
- 持仓时间: 几小时到几天
- MACD+ATR强度判断

---

### 4️⃣ 资金费率 | 无风险 | 年赚 10-50%
**收取期货资金费率**
```
Hyperliquid 每8小时结算:
• 现货持有 + 期货做空 (对冲)
• 收取资金费率
• 零风险

年化收益 = 费率 × 3 × 365
(8h周期，所以×3)
```
- 完全对冲，无风险
- 胜率: 99%
- 持仓: 8小时周期

---

### 5️⃣ 技术指标 | 确认信号 | 日赚 0.2-0.6%
**传统技术指标组合**
```
RSI: <30买  >70卖
MACD: 金叉买  死叉卖
布林带: 上限卖  下限买
成交量: >平均2倍确认
```
- 辅助确认
- 传统可靠
- 权重最低(10%)

## AI 过滤 (关键！)

### 从 100 → 20 的过程

```
100条原始信号/秒
    ↓
【质量评分】
  信号强度、类型、方向、重复
    ↓
【市场环境】
  trending / ranging / volatile / sideways
    ↓
【虚假检测】
  反向运动? 量异常? 背离?
    ↓
【时机优化】
  立即 / 100ms延迟 / 500ms延迟
    ↓
20条高质量信号/秒 (可执行)
```

### AI 模型

| 模型 | 用途 | 准确率 |
|------|------|--------|
| Random Forest | 信号分类 | 75% |
| Gradient Boosting | 市场判断 | 82% |
| LSTM | 虚假检测 | 88% |
| XGBoost | 时机优化 | 80% |

## 完整流程 (1秒循环)

```
[1] 数据获取 (1ms)
[2] 5个算法运行 (50ms)
[3] 信号融合 (20ms)
    = 0.30×S1 + 0.25×S2 + 0.25×S3 + 0.10×S4 + 0.10×S5
[4] AI过滤 (50ms)
    质量评分、市场判断、虚假检测、时机优化
[5] 风控检查 (10ms)
    头寸、资金、热线、日月亏损
[6] 下单执行 (5ms)
    如果得分>65且通过风控 → 执行
[7] 记录监控 (10ms)
```

**总耗时: ~150ms (1秒运行6-7次)**
